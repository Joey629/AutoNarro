# Naro 吉祥物设计说明

> 参考 Claude 吉祥物的**设计原则**（极简、像素块、友好、无威胁），**不复制**其造型。  
> 产品实现见 `frontend/index.html`（`#naroMascot`）与 `frontend/styles/narro.css`（`.naro-char*`）。

## 1. 设计目标

| 维度 | Claude 吉祥物 | Naro（本设计） |
|------|---------------|----------------|
| 气质 | 冷静、工具感 | 温暖、陪伴儿童讲故事 |
| 构图 | **横向**宽体 + 四足 | **竖向**书页形 + 双足 |
| 识别点 | 扁矩形 + 两侧手臂 | **顶部书签** + 圆润书页头 |
| 配色 | 单一陶土橙 | Narro 绿 + 浅腹 + **暖黄书签** |
| 隐喻 | 抽象助手 | **会动的故事书页**（叙事） |

## 2. 造型：「书芽 Naro」

竖向结构，像一本竖起的小书长出了脸和脚：

```
        ┌──┐
        │▌│  ← 书签（暖黄，叙事符号）
      ┌─┴──┴─┐
      │ ◾  ◾ │  ← 方眼（像素感，与 Claude 同级简化）
      │      │
      ├──────┤  ← 略浅的「书页腹」
      │      │
      └──┬┬──┘
        │││   ← 双脚（圆角块，非四足）
         ╰～   ← 逗号小尾巴（「讲故事」标点）
```

- **头**：圆角矩形，略宽，象征书页/封面  
- **腹**：同色阶更浅，层次类似 Claude 的扁平色块但有两层  
- **书签**：顶部小条，**Naro 独有**，避免与 Claude 剪影混淆  
- **尾**：右侧小弯钩，暗示逗号/叙述流，不喧宾夺主  

## 3. 色彩

| 部位 | Token | 用途 |
|------|--------|------|
| 主身 | `--primary` (160 84% 32%) | 头、与品牌一致 |
| 书腹 | `158 55% 42%` | 下半身体 |
| 脚/阴影 | `160 72% 26%` | 接地感 |
| 书签 | `42 96% 52%` | 点缀，儿童友好 |
| 眼 | `#1c1917` | 与 Claude 同级对比度 |

## 4. 状态（引导讲述页）

| 状态 | 动画 | 含义 |
|------|------|------|
| `idle` | 缓慢上下浮动 | 等待 |
| `listening` | 微前倾 + 眼略放大 | 在听孩子讲 |
| `thinking` | 书签轻摆、眼半闭 | 准备回应 |
| `speaking` | 弹跳 + 嘴部像素出现 + 尾轻摆 | Naro 在说话 |

## 5. 使用场景

- 仅 **Naro 引导讲述** 且已开麦时显示于图卡区上方  
- 热身寒暄与看图讲述阶段均可见  
- 不用于自由讲述、园所端、营销站
